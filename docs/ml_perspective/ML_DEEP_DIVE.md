## 1. Scope and Positioning

This document is the ML-focused companion to the integrated project documents:

- Medical context and interdisciplinary framing: ../01_MEDICAL_PROBLEM_STATEMENT.md
- Full integrated findings report: ../03_FINDINGS_REPORT.md

Core framing:

- Clinical objective: characterise dynamic lung stress at PSV flow termination.
- Engineering objective: derive robust design envelopes for Phase 3 valve development.
- ML objective: evaluate whether non-invasive waveform features (Paw + Flow) can estimate DeltaPL and provide interpretable, uncertainty-aware decision support.

ML is a translational layer, not the sole endpoint.

## 2. Data and Targets

Primary physiologic dataset:

- CCVW-ICU (public): 7 patients, 200 Hz, simultaneous Paw/Flow/Pes, Baydur-validated Pes.

Splits used in this phase:

- Local train: P01-P05 (N=222 breaths)
- Local test: P06-P07 (N=58 breaths)

Target variable:

- DeltaPL_max derived from Pes-grounded event windows around t_cycle.

## 3. Feature Engineering

### 3.1 Primary feature set (Paw + Flow only)

Primary model features (no Pes-derived predictors) include:

- f_peak, insp_dur_s, exp_dur_s

- ets_frac, ets_defaulted_flag

### 3.2 Exploratory feature extensions

Additional features explored for ablation and benchmark stress-testing:

- paw_flow_loop_area

- flow_time_product
- paw_stress_index

These were introduced to test whether richer waveform morphology improves generalisation.

## 4. Models and Evaluation Protocol

### 4.1 Models evaluated

- Mean baseline (DummyRegressor)
- Ridge baseline
- Gaussian Process regressor
- XGBoost primary (primary feature set)
- XGBoost exploratory (primary + exploratory features)

### 4.2 Evaluation design

- Leave-One-Patient-Out cross-validation (LOPO-CV) on local train
- Independent held-out local test (P06-P07)
- Regression metrics: MAE, RMSE, R2, CCC
- Uncertainty metrics for GP: predictive std and approximate PI coverage

All outputs are generated by pipeline code and exported to analysis logs for reproducibility.

## 5. Results

### 5.1 Benchmark performance

| Model | Split | MAE | RMSE | R2 | CCC | N |
|---|---|---:|---:|---:|---:|---:|
| gaussian_process | lopo_cv | 3.205 | 3.615 | 0.469 | 0.660 | 222 |
| mean_baseline | lopo_cv | 4.692 | 6.061 | -0.493 | -0.419 | 222 |
| ridge_baseline | lopo_cv | 2.114 | 2.455 | 0.755 | 0.840 | 222 |
| xgboost_exploratory | lopo_cv | 5.135 | 5.886 | -0.408 | -0.057 | 222 |
| xgboost_primary | lopo_cv | 5.151 | 5.949 | -0.438 | -0.078 | 222 |
| mean_baseline | local_test | 6.004 | 6.426 | -6.885 | 0.000 | 58 |
| ridge_baseline | local_test | 3.550 | 3.786 | -1.738 | -0.225 | 58 |
| gaussian_process | local_test | 3.717 | 4.049 | -2.131 | 0.204 | 58 |
| xgboost_primary | local_test | 3.055 | 3.338 | -1.128 | 0.015 | 58 |
| xgboost_exploratory | local_test | 3.013 | 3.341 | -1.132 | 0.092 | 58 |

Interpretation:

- Cross-patient learning is fragile with N=7.
- Ridge and GP can outperform XGBoost on LOPO-CV in this setting.
- No model demonstrates robust held-out generalisation at clinically dependable levels.

This is a rigorous negative finding, not a pipeline failure.

### 5.2 Uncertainty quantification (Gaussian Process)

Held-out local test uncertainty summary:

- Mean predictive std: 4.196 cmH2O
- Median predictive std: 4.730 cmH2O
- Approximate 95% prediction interval coverage: 84.48%

Implication:

- Uncertainty is large enough that model outputs should be interpreted as probabilistic support, not deterministic control.

### 5.3 Interpretability (permutation importance)

Top predictors in the primary XGBoost model:

1. paw_base
2. delta_paw_max
3. insp_dur_s
4. flow_decel_slope
5. f_peak

Physiologic relevance:

- These variables map to pressure state, transient strength, and cycle-shape dynamics that are mechanistically plausible correlates of lung stress.

## 7. Domain Shift Context

Simulation and VWD datasets are included as exploratory context for external validity pressure-testing:

- Simulation supports parameter-space stress checks but has known t_cycle mismatch constraints.
- VWD provides large-scale Paw/Flow morphology without Pes labels.

Therefore, domain-shift analyses are informative but not substitutes for Pes-grounded clinical generalisation.

## 8. Why Deep Learning Was Not Prioritised Here

Given cohort size and patient heterogeneity, high-capacity sequence models would likely overfit without strong pretraining alignment and substantial additional data. This phase prioritised:

- small-data model classes,
- explicit uncertainty,
- reproducible baselines,
- model interpretability.

Deep learning remains a future extension contingent on larger multi-centre data and robust label alignment.
